Space Tourism Website 🚀✨
This is a solution to the Frontend Mentor challenge. The project is a multi-page interactive website that provides an immersive experience for potential space travelers, built with React and React Router.

✨ Features
Dynamic Routing: Seamless navigation between Home, Destination, Crew, and Technology pages using react-router-dom.

Adaptive Backgrounds: The background image changes dynamically based on the current route and device size (Desktop, Tablet, Mobile) using useLocation.

Interactive Data Tabs: Content for planets, crew members, and technologies is managed via a single JSON file and updated through React state (useState).

Mobile-First Design: Fully responsive layout with a functional burger menu and side drawer for mobile users.

Modern UI Logic: Custom underlines for active tabs, CSS animations (fade-in), and clean component architecture.

🔧 Technologies Used
React (Vite) – Functional components and Hooks (useState, useLocation).

React Router v6 – Modern createBrowserRouter for clean client-side routing.

CSS3 – Advanced Flexbox, Grid, Media Queries, and @keyframes for smooth transitions.

JSON – Centralized data management for all assets and text content.

TypeScript – Ensuring type safety across the application.
